# Udemy Clone

This is a simple **Udemy homepage clone** created using **HTML5 and CSS3**.

I built this project to practice frontend development and improve my webpage design skills. This project focuses on layout creation, styling, and using Flexbox for alignment.

---

## Project Description

The website includes:

- Navigation bar  
- Search box  
- Course cards section  
- Star ratings  
- Images  
- Hover effects  

The design is inspired by the Udemy homepage and created for learning purposes only.
